# Quiz Project

This is a small Java quiz application created as a **practice exercise** to learn basic Java concepts and object‑oriented programming.

## Overview

- Written entirely in **plain Java** (no frameworks)  
- Uses a `Question` class to model quiz questions  
- `Questionservice` holds a fixed array of questions and presents them to the user  
- `Main` class drives the program, invoking the quiz and reading answers from the console  

> ✅ Completed as part of a learning/practice session – everything implemented with **core Java**  
> 🧠 Useful for beginners to understand arrays, loops, classes, and simple I/O

## How to run

```bash
javac *.java
java Main
